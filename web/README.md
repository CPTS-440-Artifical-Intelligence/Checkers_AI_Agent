# Web

Vite + React frontend for Checkers AI Agent.

## Run Locally

From `web\`:

```bat
npm.cmd run dev
```

Or use the helper script:

```bat
npm-run-dev.bat
```

Default local URL:
- `http://localhost:5173/`

## Full Stack Launcher

From repo root:

```bat
start-local-dev.bat
```

This launches both:
- FastAPI backend (`api\start-local-api.bat`)
- Vite frontend (`npm.cmd run dev` in `web\`)
